**–ü—Ä–æ–±–ª–µ–º–∞**: –ù—É–∂–Ω—ã checkpoint'—ã, —Ñ—É–Ω–∫—Ü–∏–∏, –¥–≤–∞ –ø—Ä–æ—Ö–æ–¥–∞ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è.

## 5. –û–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ –¥–∞–Ω–Ω—ã—Ö –∏–∑ —Ä–∞–∑–Ω—ã—Ö –∏—Å—Ç–æ—á–Ω–∏–∫–æ–≤

#### Snakemake:

```python

    input:
        expand("results/{sample}.txt", 
               sample=metadata['sample_id'].tolist())

    input:
        "data/{sample}.fastq"

        "process.sh {input} --condition {params.condition} > {output}"
```

#### Nextflow:

```groovy

    .map { file -> tuple(file.baseName, file) }

    .splitCsv(header: true)
    .map { row -> tuple(row.sample_id, row.condition, row.batch) }

    .set { combined_ch }
    
// –¢–µ–ø–µ—Ä—å –≤ –∫–∞–Ω–∞–ª–µ: [sample_id, fastq_file, condition, batch]

    input:
    tuple val(sample_id), path(fastq), val(condition), val(batch)
    
    output:
    path("${sample_id}.txt")
    
    script:
    """
    echo "Sample: ${sample_id}, Condition: ${condition}, Batch: ${batch}"
    process.sh ${fastq} --condition ${condition} > ${sample_id}.txt
    """
}

combined_ch | PROCESS
```
